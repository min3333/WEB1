WEB이란 무엇인가?

A web browser or browser (English: web browser or browser, cultural language: reader) is a graphical user interface-based application software that travels from a web server and communicates in two-way (navigate) and outputs HTML documents or files. Web browsers are also one of the most representative HTTP user agents. Major Web browsers include Mozilla Firefox, Google Chrome, Internet Explorer/Microsoft Edge, [1][2][3] Opera, and safari.
